Files in project:

"Person.dat" - stores login items(userID and password) of users

"AdminInformations" - stores information of admins

"ProfessorInformations" - stores information of professors

"StudentInformations" - stores information of students

"OOPquestions", "Calculusquestions" and "Physicsquestions" - store large pool of questions from subjects, respectively 

"OOP.txt", "Calculus.txt" and "Physics.txt" - store questions which will be given to students for exam

"more.txt" - stores number of questions from subjects

"Comments.txt" - stores comments

"Timer.txt" - store time that is set up by Professor
